# 自动浏览网页的python脚本

你是否有想过给自己的博客/网页有比较不错的流量呢？以下是用python写的自动运行脚本；它可以自定义网址，导入github action进行周期性计划任务。当然，最大的特点是可以自行选择sitemap.xml的url,以保证 Google Analytics的记录分析。
